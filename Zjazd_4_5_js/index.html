Zagadnienia JavaScript

Część A

  • Zad_01

    Napisz funkcję, sprawdzającą, czy dane trzy liczby tworzą „trójkę pitagorejską” (liczby nie muszą być podane w kolejności rosnącej).

    Aby wywołać funkcję Zad_01(a, b, c) z okna konsoli przeglądarki, wykonaj następujące kroki:

    1. Otwórz konsolę przeglądarki, naciskając klawisze Ctrl + Shift + J w systemie Windows lub Cmd + Option + J w systemie macOS.
    2. Wpisz poniższe polecenie w bloku kodu:
    Zad_01(a, b, c)

    Gdzie:

    • a, b, c - podstaw trzy liczby, które chcesz sprawdzić, czy tworzą trójkę pitagorejską.

    albo wpisujemy przykładowe wartości do zmiennych i nazwę funkcji: 

    const a = 3; const b = 4; const c = 5; Zad_01(a, b, c)

    Wynik działania funkcji zostanie wyświetlony w konsoli.

  • Zad_02

    Napisz funkcję wypisującą wszystkie liczby z przedziału a-b, podzielne przez c.

    Aby wywołać funkcję Zad_02(a, b, c) z okna konsoli przeglądarki, wykonaj następujące kroki:

    1. Otwórz konsolę przeglądarki, naciskając klawisze Ctrl + Shift + J w systemie Windows lub Cmd + Option + J w systemie macOS.
    2. Wpisz poniższe polecenie w bloku kodu:
    Zad_02(a, b, c)

      Upewnij się, że wartości a, b i c są zdefiniowane przed wywołaniem funkcji. Na przykład, jeśli chcesz wywołać funkcję z przedziału od 1 do 100, podzielne przez 3, wpisz:
    Zad_02(1, 100, 3)

  • Zad_04

    Napisz funkcję wypisującą w konsoli ciąg Fibonacciego o długości podanej jako parametr.

    Funkcja ta iteruje przez każdą liczbę w przedziale od 1 do n i tworzy ciąg Fibonacciego dla każdej z tych liczb. Ciąg jest wypisywany na konsoli.

    Aby wywołać funkcję Zad_04(n) z okna konsoli przeglądarki, wykonaj następujące kroki:

    1. Otwórz konsolę przeglądarki, naciskając klawisze Ctrl + Shift + J w systemie Windows lub Cmd + Option + J w systemie macOS.
    2. Wpisz poniższe polecenie w bloku kodu:
    Zad_04(n)

      Upewnij się, że wartość n jest zdefiniowana przed wywołaniem funkcji. Na przykład, jeśli chcesz wywołać funkcję dla ciągu Fibonacciego o długości 10, wpisz: 
    Zad_04(10)

  • Zad_07

    Napisz funkcję liczącą pole wybranej figury (prostokąt, trapez, równoległobok, trójkąt) z wykorzystaniem instrukcji switch. Każda figura powinna mieć osobną funkcję do liczenia pola.

    Aby wywołać funkcję Zad_07(shape, ...args) z okna konsoli przeglądarki, wykonaj następujące kroki:

    1. Otwórz konsolę przeglądarki, naciskając klawisze Ctrl + Shift + J w systemie Windows lub Cmd + Option + J w systemie macOS.
    2. Wpisz poniższe polecenie w bloku kodu, aby obliczyć pole prostokąta o bokach 5 i 10:
    Zad_07('rectangle', 5, 10)
    1. Wpisz poniższe polecenie w bloku kodu, aby obliczyć pole trapezu o podstawach 5 i 10 oraz wysokości 8:
    Zad_07('trapezoid', 5, 10, 8)
    1. Wpisz poniższe polecenie w bloku kodu, aby obliczyć pole równoległoboku o bokach 5 i 10:
    Zad_07('parallelogram', 5, 10)
    1. Wpisz poniższe polecenie w bloku kodu, aby obliczyć pole trójkąta o podstawie 5 i wysokości 10:
    Zad_07('triangle', 5, 10)
